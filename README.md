# Digital QC

## Overview

Digital Quality Control System for Green Concrete developed using Python and Dash.

This application helps civil engineers and researchers analyze and evaluate concrete experimental results according to multiple international design codes.

## Features

- Compression Strength Analysis (7, 28, and 56 Days)
- Tensile Strength Analysis
- Flexural Strength Analysis
- Automatic Quality Control Evaluation
- Multiple Design Codes Support:
  - ECP 203
  - ACI 318
  - BS 8110
  - SBC 304
- Interactive Dashboard
- Automated PDF Report Generation
- Excel Data Import

## Technologies Used

- Python
- Dash
- Plotly
- Pandas
- NumPy
- Matplotlib
- ReportLab

## How to Run

Install required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

## Application Purpose

This project was developed to support research and quality control processes for green concrete mixtures utilizing sustainable materials and digital engineering tools.
